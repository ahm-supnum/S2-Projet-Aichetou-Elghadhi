# S2-Projet-Aichetou-Elghadhi

**Nom:** Aichetou elghadhi
**Matricule:**25096
**Titre du projet:** Internet : Progrès Technologique ou Addiction Moderne

**Description:**
Ce projet explore l'impact de l'internet dans le monde contemporain.
Nous étudions si l'internet représente un véritable progrès technologique 
ou une source croissante d'addiction pour les utilisateurs.
Le projet contient une présentation, une analyse statistique et une méthodologie.
